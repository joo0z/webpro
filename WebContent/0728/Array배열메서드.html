1. indexOf/ lastIndexOf

arr = ["사과", "체리", "양파", "짜장면", "딸기", "사과", "복숭아", "키위", "망고", "사과"]
indexOf("사과") => 처음나온 사과의 위치
indexOf("사과", 3) => 3번째 사과 이후의 사과위치
lastIndexOf("사과"); => 마지막 사과위치



2. push/ pop

arr = ["사과", "체리", "양파", "짜장면", "딸기", "사과", "복숭아", "키위", "망고", "사과"]
push => 배열의 마지막에 추가
pop => 배열의 마지막 요소 제거
