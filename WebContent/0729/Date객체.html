1. Date객체 생성

- new Date()
- new Date(98765678978) 1979년1월1일 0시를 기준으로 얼마나 지났는지 알려준다.
- new Date("2010.02.23")
- new Date(2010,01,23,12,30,12)
- 월은 0 부터 시작 => -1해줘야 한다.


2. Date객체 생성

- now = new Date()
- now.toLocaleString() : 날짜와 시간
- now.toLocaleDateString() : 날짜
- now.toLocaleTimeString() : 시간
